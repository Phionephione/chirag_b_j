# Chirag B J - Dynamic React Portfolio

This is a dynamic and interactive personal portfolio website for Chirag B J, built with React, TypeScript, and Tailwind CSS. It showcases skills, certifications, projects, and professional background in a modern, user-friendly interface.

## Features

- **Interactive UI:** A dynamic grid background and content cards that light up and react to the user's cursor movement.
- **Dynamic Header:** An animated typing effect that cycles through different professional titles.
- **Smooth Animations:** Subtle fade-in animations for all content sections as they appear.
- **Component-Based:** Built with reusable React components for maintainability and scalability.
- **Fully Responsive:** Designed to look great on all devices, from mobile phones to desktops.
- **Deployment Ready:** Configured with Vite for a fast development experience and optimized builds, ready for easy deployment on Vercel.

## Getting Started

To run this project locally, follow these steps:

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or later recommended)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```
    This will start the Vite development server, and you can view the site at `http://localhost:5173` (or another port if 5173 is busy).
